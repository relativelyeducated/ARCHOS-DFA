# Data Directory

## Structure

### raw/
Original, unmodified datasets:
- `icecube/` - IceCube neutrino observatory data
- `grok-exports/` - Grok AI conversation exports
- `claude-exports/` - Claude AI conversation exports

### processed/
Processed and cleaned data:
- `energy-bins/` - Neutrino events grouped by energy
- `d2-calculations/` - Fractal dimension calculations

## Data Sources

### IceCube Data
- **Source**: IceCube Neutrino Observatory
- **Format**: `.dat` files with energy and zenith angle
- **Events**: 336,516 neutrino detections
- **Energy Range**: 100 GeV - 100+ TeV

### Conversation Exports
- **Grok**: 40 conversations, Feb-Nov 2025
- **Claude**: [To be added]
- **Format**: JSON and Markdown
